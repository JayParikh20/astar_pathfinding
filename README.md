# A-star Path Planning using ROS and Python

### System Info
* Ubuntu 20.04 LTS
* ROS Noetic
*  Python 3.8.10

### To run the project use
Source your ros workspace first, and build package if not done
```
cd Desktop/ros1_ws/
source devel/setup.bash
catkin_make
roslaunch astar astar.launch
```
